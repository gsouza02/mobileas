PARA INICIAR O APP.

No computador:

Possuir NodeJS instalado.
No terminal, verificar se o diretório condiz com a pasta mobileas.
Se estiver correto: npm install.
Após instalação, npm start.

No celular:

verificar se o celular está na mesma rede do computador.
Baixar o app Expo Go (iOs e Android)
Escanear o qr code.
